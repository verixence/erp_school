## üéì Best Practices

### **DO:**
‚úÖ Use theme variables (`schoolTheme.colors.primary.main`)
‚úÖ Follow semantic color system (success/warning/error/info)
‚úÖ Keep gradients subtle (10% brightness difference max)
‚úÖ Use Inter font for all text
‚úÖ Maintain white backgrounds
‚úÖ Test on both iOS and Android

### **DON'T:**
‚ùå Hardcode colors (`'#FF6B35'`)
‚ùå Use cream backgrounds
‚ùå Create bright gradients (>20% difference)
‚ùå Use System font
‚ùå Add new random colors
‚ùå Skip accessibility testing
